Standard Production Template

This template initializes a new project environment linked to the kicad-libs repository.

Pre-configured Settings:

  • Path Variables: Links to local 3D models and datasheets.
  • Library Tables: Global symbol and footprint libraries loaded.
  • DRC Rules: Standard manufacturing constraints (Min track 0.2mm, etc.).
  • Page Layout: Custom company title block applied.

Usage: After creating the project, copy required hierarchical sheets from the 'modules' directory into your project folder manually.
